Robotic Process Automation of UI

Built a Cognitive Intelligent system that implements Robotic Process Automation for automated data entry through UI. Using Optical Character Recognition, required details are read from file sources which are required for data entry. Using OpenCV, Image Processing was used to identify the various fields in the UI which required data entry. Using automation techniques, keyboard and mouse strokes are mimicked to input the required data into the desired fields. Copious amount of files are processed in a small amount of time. Data entry validation can be done manually by checking the database. Speed can also be changed to slow down the automation process to view the data entry in UI.
